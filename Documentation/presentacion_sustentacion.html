Objetivos

¿Qué se propuso?

🎯 Objetivo General

Desarrollar un OVA que integre un modelo geoestadístico 3D del Vshale, construido con scripts Python públicos, para fortalecer competencias de modelado en estudiantes de geociencias e ingeniería de petróleos.

📌 Objetivos Específicos
  • 1. Estandarizar registros de pozo públicos (NOPIMS) mediante Python
  • 2. Calcular Vshale con la ecuación de Larionov (formaciones terciarias)
  • 3. Construir modelo 3D mediante IDW con control de anisotropía
  • 4. Diseñar estrategia pedagógica audiovisual para replicabilidad académica

Resultados – Vshale

Cálculo del Vshale

Vshale = 0.083 × (23.7 × GR_norm − 1)     [Larionov – Formaciones Terciarias]
📥 Proceso
  • Entrada: Gamma Ray (GR) crudo de pozos
  • Paso 1: Normalización del GR (GRmin / GRmax por pozo)
  • Paso 2: Aplicación ecuación Larionov (formaciones modernas)
  • Salida: Curva Vshale continua (0 a 1) por pozo
📤 Interpretación
  • Vshale cercano a 0 → Roca limpia (Arena/reservorio)
  • Vshale cercano a 0.5 → Limolita / zona transicional
  • Vshale cercano a 1 → Lutita / sello
Resultados – Modelo 3D

Interpolación IDW + Visualización 3D

▪ Segmentación Litológica (Blocking)
  • Agrupa miles de lecturas en bloques geológicos homogéneos
  • Parámetros: Window=5, Umbral=0.05
  • Reduce ruido eléctrico · resalta capas estratigráficas reales
  • Cada bloque recibe un Segmento_ID único
🗺 IDW Anisotrófico
  • Malla estructurada (StructuredGrid) X–Y–Z
  • Factor de anisotropía: 0.05 en Z — fuerza continuidad lateral
  • Evita artefactos visuales (“ojos de buey”)
  • Sliders para cortes e isosuperficies con PyVista

Conclusiones y Recomendaciones

  • OVA documentado con flujo computacional, manual y serie audiovisual, cumpliendo objetivos instruccionales
